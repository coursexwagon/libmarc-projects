    58→## Page Building Conventions (MANDATORY for consistency)
    59→1. **Page structure**: Use `PageHero` at top (dark bg with breadcrumb + title + description), then content sections, then `CTABand` at bottom.
    60→2. **Use shared components**: `SectionHeading`, `Reveal`, `Counter`, `CTABand`, `Breadcrumbs` from `@/components/site/sections`.
    61→3. **Image import**: `import Image from "next/image"`. Use `<Image src="..." fill className="object-cover" sizes="..." />` with aspect ratio containers.
    62→4. **Buttons**: yellow CTA = `bg-primary text-primary-foreground hover:bg-primary/90 font-bold uppercase tracking-wide`. Use `<Button asChild>...<Link></Button>`.
    63→5. **Dark sections**: `bg-foreground text-background` with hazard stripe accent (`<div className="hazard-stripe h-1.5 w-full" />`).
    64→6. **Spacing**: section padding `py-20 lg:py-28`. Container: `container mx-auto px-4`.
    65→7. **Icons**: lucide-react. Yellow accent icons `text-primary`.
    66→8. **Cards**: hover `hover:border-primary hover:shadow-xl transition-all`.
    67→9. **Dynamic routes**: use `generateStaticParams` + `notFound()` for `[slug]` pages. Import data from `@/lib/site-data`.
    68→10. **Contact form**: build a client form that POSTs to `/api/contact` (create the API route too).
    69→11. **Markdown blog content**: render `blogPost.content` (has `## ` headings and `\n\n` paragraphs) — split on `\n\n`, render headings vs paragraphs.
    70→12. Keep pages responsive (mobile-first). Test with `bun run lint`.
