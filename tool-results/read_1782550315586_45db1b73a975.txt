    58‚Üí    58‚Üí## Page Building Conventions (MANDATORY for consistency)
    59‚Üí    59‚Üí1. **Page structure**: Use `PageHero` at top (dark bg with breadcrumb + title + description), then content sections, then `CTABand` at bottom.
    60‚Üí    60‚Üí2. **Use shared components**: `SectionHeading`, `Reveal`, `Counter`, `CTABand`, `Breadcrumbs` from `@/components/site/sections`.
    61‚Üí    61‚Üí3. **Image import**: `import Image from "next/image"`. Use `<Image src="..." fill className="object-cover" sizes="..." />` with aspect ratio containers.
    62‚Üí    62‚Üí4. **Buttons**: yellow CTA = `bg-primary text-primary-foreground hover:bg-primary/90 font-bold uppercase tracking-wide`. Use `<Button asChild>...<Link></Button>`.
    63‚Üí    63‚Üí5. **Dark sections**: `bg-foreground text-background` with hazard stripe accent (`<div className="hazard-stripe h-1.5 w-full" />`).
    64‚Üí    64‚Üí6. **Spacing**: section padding `py-20 lg:py-28`. Container: `container mx-auto px-4`.
    65‚Üí    65‚Üí7. **Icons**: lucide-react. Yellow accent icons `text-primary`.
    66‚Üí    66‚Üí8. **Cards**: hover `hover:border-primary hover:shadow-xl transition-all`.
    67‚Üí    67‚Üí9. **Dynamic routes**: use `generateStaticParams` + `notFound()` for `[slug]` pages. Import data from `@/lib/site-data`.
    68‚Üí    68‚Üí10. **Contact form**: build a client form that POSTs to `/api/contact` (create the API route too).
    69‚Üí    69‚Üí11. **Markdown blog content**: render `blogPost.content` (has `## ` headings and `\n\n` paragraphs) ‚Äî split on `\n\n`, render headings vs paragraphs.
    70‚Üí    70‚Üí12. Keep pages responsive (mobile-first). Test with `bun run lint`.
